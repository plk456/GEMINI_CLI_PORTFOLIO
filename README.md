# Prodduturu Likith Kumar - Portfolio Project

> **Created by Prodduturu Likith Kumar using Gemini CLI**

This project is a modern, responsive portfolio website built using React and TypeScript, based on the professional experience and skills extracted from `RESUME.pdf`.

## Project Structure

- `portfolio/`: The React + TypeScript + Vite application.
- `python/`: Existing Python analytics and data processing scripts.
- `RESUME.pdf`: The source document for the portfolio content.
- `.gitignore`: Configured to ignore `.env` and other sensitive files.

## Portfolio Features

- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Dark Theme:** A sleek, modern aesthetic using a dark color palette.
- **Sections:**
  - **Hero:** Introduction with a professional bio.
  - **Education:** Academic background and credentials.
  - **Technical Skills:** Categorized list of languages, AI frameworks, tools, and databases.
  - **Featured Projects:** Showcasing key work like the Intelligent RAG Chatbot and GPT-2 Medical Assistant.
  - **Achievements:** Highlighting competition successes.
  - **Contact:** Quick links to professional profiles (LinkedIn, GitHub, Kaggle).

## Getting Started

To run the portfolio locally:

1. Navigate to the portfolio directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Technologies Used

- **Frontend:** React, TypeScript
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (Custom properties for theming)
- **Deployment:** Ready for platforms like Vercel or Netlify.
